# Portfolio

A simple, elegant one-pager portfolio website with smooth animations.

## Features

- **Overview Section**: Hero section with introduction and call-to-action buttons
- **Professional Experience**: Timeline view of career progression
- **Capabilities & Skills**: Showcase of technical skills with animated progress bars
- **Smooth Animations**: Fade-in effects and interactive hover animations
- **Responsive Design**: Optimized for desktop viewing

## How to View

Simply open `index.html` in a web browser or serve it with a local server:

```bash
python3 -m http.server 8080
```

Then navigate to `http://localhost:8080` in your browser.

## Customization

The content is placeholder data. Update the following in `index.html`:

- Personal name and title in the hero section
- Professional experience details in the timeline
- Skills and capabilities in the skills section
- Social media links in the footer

## Technologies Used

- HTML5
- CSS3 (with CSS animations)
- Vanilla JavaScript (Intersection Observer API for scroll animations)
